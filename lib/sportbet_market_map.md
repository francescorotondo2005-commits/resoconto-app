# SPORTBET — Dizionario Mercati (Fastbet Tendina Verticale)
# Formato: TITOLO_SPORTBET → NOME_INTERNO
# 
# Convenzioni:
#   - "SQUADRA  1" = Casa (doppio spazio nel sito!)
#   - "SQUADRA  2" = Ospite
#   - "T.R."       = Totale Reale (senza buoni giocatori)
#   - "INCL.TS"    = Inclusi i Tiri Sbagliati
#   - "TEAM 1"     = Squadra Casa
#   - "TEAM 2"     = Squadra Ospite

## GOL TOTALI
U/O 0.5               → UNDER/OVER 0,5 GOL
U/O 1.5               → UNDER/OVER 1,5 GOL
U/O 2.5               → UNDER/OVER 2,5 GOL
U/O 3.5               → UNDER/OVER 3,5 GOL
U/O 4.5               → UNDER/OVER 4,5 GOL
U/O 5.5               → UNDER/OVER 5,5 GOL

## GOL SQUADRA (nota: il numero della linea è nella parte della label, es "UNDER 1.5")

U/O 1.5 SQUADRA  1    → UNDER/OVER 1,5 GOL CASA
U/O 2.5 SQUADRA  1    → UNDER/OVER 2,5 GOL CASA
U/O 0.5 SQUADRA  2    → UNDER/OVER 0,5 GOL OSPITE
U/O 1.5 SQUADRA  2    → UNDER/OVER 1,5 GOL OSPITE
